## 🚀 Deployment Steps
1. Get your `SESSION_ID` by scanning the QR or using a pairing code. Then go to `WhatsApp > Three dots > Linked Devices`
2. Get your `MONGODB_URI` from [MongoDB](https://www.mongodb.com/) *(Optional)*
3. Fork and Star ⭐ the repository
4. Set your environment variables (see `Setting.js`)
5. Deploy on your preferred platform

---

## ⚙️ Key Variables
| Variable | Description |
|---|---|
| `SESSION_ID` | Your WhatsApp session |
| `OWNER_NUMBER` | Your phone number (international format, no +) |
| `MONGODB_URI` | MongoDB connection string |
| `BOT_NAME` | Bot display name (default: Jarvis-Md) |
| `OWNER_NAME` | Your name (default: Vince) |
| `PREFIX` | Command prefix (default: .) |
| `WORKTYPE` | `private` or `public` |
| `OPENAI_API_KEY` | For AI/ChatGPT features |
